ArsyaCustomRTP - Purpur 1.20.4

Fitur:
- Biaya RTP berdasarkan persentase saldo
- Maksimal potongan bisa diatur
- Minimal saldo bisa diatur
- Bisa dipakai dari DeluxeMenus dengan command /customrtp <worldkey>
- Menggunakan Vault economy

Contoh DeluxeMenus:
  left_click_commands:
    - '[player] customrtp overworld'
    - '[close]'

Command plugin:
- /customrtp overworld
- /customrtp nether
- /customrtp end
- /customrtp reload

Catatan:
- Karena environment ini tidak punya Maven + dependency server API, saya buatkan project source siap compile.
- Compile di PC/VPS kamu dengan Maven, lalu ambil file JAR hasil build.
